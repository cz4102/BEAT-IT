# BEAT-IT
BEAT-IT: A Wearable Heart Rate Monitor with Coronary Artery Disease Risk Prediction via Android Application using Logistic Regression is a device developed as a undergrad thesis for our Computer Engineering Bachelor's Degree. It is to provide proactive awareness for the risk of coronary artery disease or heart disease.

For the IoT platform, we used the Arduino Cloud IoT which also served as our user interface for the integration and use of the app.

Database: Firebase
Machine Learning: Trained using Google Colaboratory

Hardware: Seeed Studio XIAO ESP32-C3, MAX30102 Breakout board, 3.3 V Lithium Polymer Battery
Enclosure/Case Material: Custom 3D printed using TPU45A and PLA+ Filament (Backed by a Material Science Expert from the Department of Science and Technology, Material Science Division)
